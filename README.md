### Catálogo de Sistemas Interactivo
8 categorías de sistemas con tarjetas premium:

1. **Corredizo** - Sistemas deslizantes de alto rendimiento
2. **Abatibles** - Apertura tradicional con tecnología moderna
3. **Fijos** - Panoramas ininterrumpidos de luz natural
4. **Plegable** - Flexibilidad total para abrir espacios completos
5. **Domos** - Entrada de luz cenital con diseño arquitectónico
6. **Cristal Templado** - Seguridad y resistencia con máxima transparencia
7. **Acoples** - Uniones estructurales de alta precisión
8. **Especiales** - Soluciones personalizadas para proyectos únicos

#### Características de las Tarjetas
- Sin bordes visibles
- Sombras suaves
- Diseño flotante
- Bordes redondeados
- Galería automática de imágenes con transición suave
- Hover con elevación, zoom ligero y cambio de iluminación
- Totalmente responsive

### Cotizador Visual Inteligente

#### Selector de Categoría
Botones visuales para elegir el tipo de sistema.

#### Barras Dinámicas
- **Ancho**: Rango 50-800 cm con visualización en tiempo real
- **Alto**: Rango 50-600 cm con visualización en tiempo real

#### Croquis Dinámico Inteligente
El croquis cambia en tiempo real según:
- La categoría seleccionada
- Las medidas de ancho y alto
- Escalado proporcional automático

**Tipos de croquis:**
- **Corredizo**: Muestra hojas corredizas representativas
- **Abatible**: Muestra hojas con apertura lateral
- **Fijo**: Muestra panel fijo panorámico
- **Plegable**: Muestra sistema plegable segmentado (4 paneles)
- **Domo**: Muestra representación tipo domo arquitectónico
- **Cristal Templado**: Muestra panel templado con marcas de seguridad

#### Casos Especiales
Para **Acoples** y **Especiales**:
- No se muestra el croquis
- Se muestra tarjeta premium con ilustración creativa
- Mensaje: "Este proyecto requiere al cerebro del equipo"
- Botón directo al Cotizador Inteligente

#### Datos en Tiempo Real
- Ancho seleccionado
- Alto seleccionado
- Área calculada automáticamente (en m²)
- Vista previa gráfica del sistema

### Cotizador Inteligente (IA)
Sección destacada que comunica:
- Asesoría especializada
- Cálculo avanzado
- Proyectos complejos
- Soluciones personalizadas

**Características destacadas:**
- Análisis automático de medidas

- Optimización de costos en tiempo real
- Asesoría especializada incluida

## 🚀 Cómo Usar

1. Abre el archivo `index.html` en tu navegador
2. Explora las diferentes secciones haciendo scroll
3. Interactúa con las tarjetas de sistemas
4. Usa el cotizador visual:
   - Selecciona una categoría
   - Ajusta ancho y alto con los sliders
   - Observa cómo cambia el croquis en tiempo real
5. Para proyectos especiales, haz clic en "Consultar Cotizador Inteligente"
